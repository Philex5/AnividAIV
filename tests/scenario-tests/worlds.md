# 世界设定 (Worlds) - 场景测试

## 测试目标

验证世界 (World) 的创建、角色关联以及世界主页的展示功能。

## 前置条件

- [ ] 服务已启动
- [ ] 已登录测试账号
- [ ] 拥有至少一个 OC 角色（用于关联到世界）

---

### 场景 1: 创建新世界

**目标**: 验证用户能否创建一个新的世界观背景。

#### 测试步骤
1. [ ] 进入 `/worlds/create` 页面
2. [ ] 填写世界基本信息：
    - 世界名称
    - 背景描述
    - 主题颜色
3. [ ] 上传世界封面图
4. [ ] 点击"创建"

#### 预期结果
- [ ] 世界成功创建并保存到数据库
- [ ] 自动跳转到新创建的世界详情页 `/worlds/[uuid]`
- [ ] 页面显示所填写的信息和封面图

---

### 场景 2: 将角色关联到世界

**目标**: 验证 OC 角色与世界之间的从属关系。

#### 测试步骤
1. [ ] 打开一个已有的 OC 角色编辑页面
2. [ ] 在设置中找到"所属世界"选项
3. [ ] 从下拉列表中选择刚才创建的世界
4. [ ] 保存角色信息
5. [ ] 访问世界主页 `/worlds/[uuid]`

#### 预期结果
- [ ] 角色详情页显示所属世界的链接
- [ ] 世界主页的"角色列表"中出现了该 OC 角色

---

### 场景 3: 世界主页展示

**目标**: 验证世界主页的沉浸式展示效果。

#### 测试步骤
1. [ ] 访问一个公开的世界页面
2. [ ] 检查页面元素：
    - 世界封面图
    - 世界观介绍
    - 关联的角色列表
    - 相关的创作（画作/视频）

#### 预期结果
- [ ] 布局符合设计，展示世界观氛围
- [ ] 角色点击可跳转到角色详情
- [ ] 如果该世界下有作品，应能正确展示
