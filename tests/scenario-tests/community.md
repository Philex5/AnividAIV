# 社区互动 - 场景测试

## 测试目标

验证社区作品浏览、点赞、收藏等互动功能。

## 前置条件

- [ ] 服务已启动
- [ ] 已登录测试账号
- [ ] 社区已有一些公开作品

## 核心测试场景

### 场景 1: 浏览社区作品
- [ ] 打开社区页面 `/community`
- [ ] 验证作品瀑布流布局
- [ ] 滚动加载更多作品
- [ ] 验证图片懒加载

### 场景 2: 筛选和排序
- [ ] 按类型筛选: 全部/图片/视频/OC
- [ ] 按排序: 最新/最热/最多点赞
- [ ] 验证筛选结果正确

### 场景 3: 搜索作品
- [ ] 输入关键词搜索
- [ ] 验证搜索结果
- [ ] 清空搜索

### 场景 4: 查看作品详情
- [ ] 点击作品打开详情模态窗口
- [ ] 查看作品信息: Prompt、参数、作者
- [ ] 关闭详情

### 场景 5: 点赞作品
- [ ] 点击点赞按钮
- [ ] 验证点赞数增加
- [ ] 点赞按钮状态改变（已点赞）
- [ ] 再次点击取消点赞

### 场景 6: 收藏作品
- [ ] 点击收藏按钮
- [ ] 验证收藏成功
- [ ] 在"我的收藏"查看收藏的作品
- [ ] 取消收藏

### 场景 7: 分享作品
- [ ] 点击分享按钮
- [ ] 复制分享链接
- [ ] 验证链接可访问

### 场景 8: 我的作品管理
- [ ] 打开"我的作品"页面 `/my-artworks`
- [ ] 查看自己的作品
- [ ] 修改作品可见性（公开/私密）
- [ ] 删除作品

### 场景 9: OC 展示
- [ ] 浏览公开的 OC 角色
- [ ] 查看 OC 详情
- [ ] 查看 OC 的创作历史

### 场景 10: 社区规则
- [ ] 验证不合规内容被隐藏
- [ ] 举报功能（如果有）
