# AI 视频生成器 - 场景测试

## 测试目标

验证 AI 视频生成器的各种使用场景，包括文字生成视频、图片生成视频、不同模型和参数组合等。

## 前置条件

- [ ] 服务已启动
- [ ] 已登录测试账号
- [ ] 测试账号有足够积分（视频生成消耗较多，建议 ≥200 MC）
- [ ] 准备测试图片（用于图片转视频）

## 核心测试场景

### 场景 1: 文字生成视频（Text to Video）
- [ ] 输入 Prompt: `a cute anime girl dancing`
- [ ] 选择模型: `kling-1.6-pro`
- [ ] 选择时长: `5 秒`
- [ ] 选择比例: `16:9`
- [ ] 查看报价
- [ ] 确认生成
- [ ] 验证视频质量和时长

### 场景 2: 图片生成视频（Image to Video）
- [ ] 上传参考图片
- [ ] 输入运动 Prompt: `the character is waving hand`
- [ ] 选择模型和参数
- [ ] 生成视频
- [ ] 验证视频保持图片内容并添加运动

### 场景 3: 不同模型对比
测试以下模型:
- [ ] `kling-1.6-pro` - 高质量
- [ ] `kling-1.6` - 标准质量
- [ ] `wan-video-1.0` - 另一模型

验证质量差异和费用差异

### 场景 4: 不同时长测试
- [ ] 5 秒视频
- [ ] 10 秒视频
- [ ] 验证费用随时长增加

### 场景 5: 不同比例测试
- [ ] 16:9 (横屏)
- [ ] 9:16 (竖屏)
- [ ] 1:1 (方形)

### 场景 6: Prompt 优化
- [ ] 使用简短 Prompt
- [ ] 点击"优化 Prompt"
- [ ] 验证优化结果更专业

### 场景 7: 视频生成状态轮询
- [ ] 创建视频任务
- [ ] 观察状态变化: pending → processing → completed
- [ ] 验证预计完成时间显示
- [ ] 完成后自动刷新

### 场景 8: 视频结果操作
- [ ] 播放视频
- [ ] 下载视频
- [ ] 分享到社区
- [ ] 重新生成（Remix）

### 场景 9: 错误处理
- [ ] 积分不足测试
