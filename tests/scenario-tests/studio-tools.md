# OC 应用 (Studo Tools) - 场景测试

## 测试目标

验证基于 OC 角色的扩展应用功能，包括 AI 手办生成器、AI 贴纸生成器和 Art Prompt 生成器。

## 前置条件

- [ ] 服务已启动
- [ ] 已登录测试账号
- [ ] 拥有至少一个 OC 角色
- [ ] 有足够积分

---

### 场景 1: AI 手办生成器 (Action Figure Generator)

**目标**: 验证为 OC 角色生成精美手办效果图的功能。

#### 测试步骤

1. [ ] 打开 `/ai-action-figure-generator`
2. [ ] 点击"选择角色"并选中一个 OC
3. [ ] 在模板列表中选择一个手办模板（如：Q版、精细手办等）
4. [ ] 点击"生成"按钮
5. [ ] 等待生成完成

#### 预期结果

- [ ] 页面正确加载 OC 角色信息
- [ ] 模板预览图显示正常
- [ ] 生成过程中显示进度
- [ ] 生成的结果图保留了 OC 的核心特征（发色、瞳色等），并呈现手办质感
- [ ] 积分正确扣减

---

### 场景 2: AI 贴纸生成器 (Sticker Generator)

**目标**: 验证为 OC 角色生成表情包/贴纸的功能。

#### 测试步骤

1. [ ] 打开 `/ai-sticker-generator`
2. [ ] 选中一个 OC 角色
3. [ ] 输入或选择一个表情主题（如：开心、生气、惊讶）
4. [ ] 点击"生成"
5. [ ] 验证生成结果

#### 预期结果

- [ ] 生成的图片具有贴纸风格（通常带有白边，背景简单或透明感）
- [ ] 角色表情与选择的主题一致
- [ ] 积分扣减正确

---

### 场景 3: Art Prompt 生成器

**目标**: 验证将简单描述转化为高质量画作 Prompt 的功能。

#### 测试步骤

1. [ ] 打开 `/art-prompt-generator`
2. [ ] 输入基础创意关键词
3. [ ] 选择艺术风格和氛围选项
4. [ ] 点击"生成高级 Prompt"
5. [ ] 复制生成的 Prompt 并尝试在 AI 动漫生成器中使用

#### 预期结果

- [ ] 生成的 Prompt 词汇丰富且专业
- [ ] 包含光影、构图、渲染等维度的修饰词
- [ ] 提供"一键复制"功能且工作正常
