# FEAT-WORLDS 世界观社交用例

关联需求：FEAT-WORLDS（docs/1-specs/PRD.md）

## 主路径

1. 登录用户进入公开世界观详情页
2. 执行点赞/收藏/分享
3. 断言计数更新且状态正确
4. 发布评论并在列表顶部可见

## 错误与边界

1. 未登录用户点击点赞/收藏/评论/分享
   - 期望返回 `AUTH_REQUIRED`
2. 访问私有世界观社交接口
   - 期望返回 `FORBIDDEN`
3. 评论内容为空或超过 1000 字符
   - 期望返回 `VALIDATION_FAILED`

## 回归

1. 删除评论后计数同步更新且评论不可见
2. 重复点赞/收藏不重复增加计数
3. 分享记录不会影响点赞/收藏状态
