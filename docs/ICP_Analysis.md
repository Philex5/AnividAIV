# AnividAI - 完美用户画像 (ICP) 分析

基于 AnividAI 的产品文档、架构设计及功能特性，我们定义了本平台的**完美用户画像 (ICP)**。

## 1. 执行摘要

AnividAI 的“完美用户”是深耕 ACGN（动漫、漫画、游戏、小说）文化的**创意故事讲述者或内容创作者**。他们拥有强烈的愿望去可视化自己的想象力——特别是原创角色（OC）——但缺乏专业绘画技能或手工绘制的时间。相比于钻研复杂的技术参数，他们更看重**角色一致性**（在不同画面/视频中保持角色形象统一）、**叙事效率**和**变现潜力**。

---

## 2. 人口统计学特征

*   **年龄：** 15 - 35 岁（Z世代 & 千禧一代）。
*   **职业：** 学生、业余作家、独立游戏开发者、虚拟主播 (VTuber)、自媒体创作者、社群管理员。
*   **地域：** 全球受众，主要集中在动漫文化流行地区（东亚、北美、欧洲、东南亚）。
*   **语言：** 多语言用户，习惯使用中文或英文界面。
*   **收入水平：**
    *   **第1层级 (免费/基础版)：** 可支配收入有限的学生或业余爱好者。
    *   **第2层级 (专业/高级版)：** 愿意为生产力工具和角色一致性付费（$20-$50/月）的内容创作者或专业人士。

## 3. 心理特征

*   **兴趣：** 动漫、漫画、轻小说、同人小说、角色扮演 (RP)、虚拟主播、角色设计、世界观构建。
*   **价值观：** 创造力、自我表达、社区认同、效率、“颜值即正义”。
*   **性格：** 想象力丰富，内向但在网络社交中活跃（Discord, Reddit, Twitter/X, Bilibili），对特定圈子充满热情。
*   **痛点：**
    *   *“我脑海里有个很棒的故事/角色，但我手残不会画。”*
    *   *“找画师约稿太贵了，而且排期很长。”*
    *   *“其他的 AI 工具生成的动漫图都是随机脸，我想要**我的**角色长得一样。”*
    *   *“我想做漫画或PV，但学 AE/PS 太难了。”*

## 4. 行为特征

*   **技术能力：** 中等。能熟练使用 Web 应用和社交媒体，但不一定是 AI 工程师或 Stable Diffusion 专家。相比复杂的提示词工程，更喜欢“一键生成”的解决方案。
*   **平台使用：** 重度手机和电脑用户。活跃于 TikTok, Twitter (X), Instagram, YouTube Shorts, Bilibili, 小红书。
*   **购买行为：**
    *   通常从 **免费版** 开始，测试“角色一致性”效果。
    *   当触及使用额度限制或需要特定高级功能（如视频生成、高清下载）时，升级为 **订阅会员**。
    *   在进行特定项目或创作灵感爆发时，倾向于冲动购买 **加油包 (Meow Packs)**。

## 5. 核心应用场景 (Jobs to be Done)

### A. “世界架构师” & “轻小说作者” (Indie Storyteller)
*   **目标：** 将小说、跑团（D&D）或脑洞故事可视化，制作成漫画或 PV 展示给读者。
*   **行为：** 使用 **Story Lab** 上传剧本。使用 **Manga Studio** 生成连载条漫。
*   **价值：** “无需昂贵的画师，我也能让我的小说拥有‘动画化’的待遇。”

### B. “虚拟网红” (潜力 VTuber/博主)
*   **目标：** 在不露脸的情况下，建立一致的虚拟人设以实现社交媒体增长。
*   **行为：** 创建 OC。生成 **手办 (Action Figures)** 或 **贴纸 (Stickers)** 用于发帖。使用 **视频生成器** 制作短视频内容。
*   **价值：** “无需组建动画团队，我也能为我的 IP 持续产出内容。”

### C. “创作者商人” (Marketplace Trader)
*   **目标：** 通过出售优质素材获利或获得社区声望。
*   **行为：** 精心调教 Prompt 和 OC 设子。在 **Asset Store** 上架自己的资源。
*   **价值：** “我的审美和提示词技巧可以变现，这让我更有动力创作。”

## 6. 需求-功能匹配矩阵

| ICP 需求 | AnividAI 解决方案 |
| :--- | :--- |
| **“我不会画画。”** | **Character Studio：** 带有视觉引导的文字转角色功能，包括一句话快速生成。 |
| **“我需要角色长得一样。”** | **角色系统 & Reference Sheets：** 底层数据库在每次生成时自动注入 OC 特征，并提供三视图生成。 |
| **“我有故事，想做成漫画/视频。”** | **Story Lab & Manga Studio：** 从剧本到分镜再到成品的完整流水线。 |
| **“我想火/想分享。”** | **社区 & 徽章系统：** 游戏化的成就体系和优化的社交分享卡片。 |
| **“我想赚钱。”** | **Creator Marketplace：** 允许出售 Prompt 和素材赚取积分。 |

## 7. 获客渠道

*   **社交媒体：** Twitter/X (绘画圈), TikTok (AI 特效趋势), Pinterest (角色灵感), 小红书 (OC设子)。
*   **社区：** Discord 服务器 (Midjourney, Stable Diffusion, RP 群组), Reddit (r/worldbuilding, r/characterdrawing), 百度贴吧。
*   **网红营销：** VTuber 展示“我如何用 AI 制作我的同人图”。
