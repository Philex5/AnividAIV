## 画廊重构：多维角色聚光灯 (OC Spotlight Gallery)

不再是简单的缩图网格，画廊将进化为展示 OC “灵魂深度”的舞台。采用 **“中心立绘 + 四角维度数据”** 的创新布局，直接复用角色详情页的核心组件和数据结构。

### 1. 核心设计：四角聚光灯布局 (Four-Corner Spotlight)

每个展示的角色卡片将采用响应式聚光灯布局，中间为人物立绘，四个角落分别对应角色详情页的四个核心信息维度：

- **中心：视觉之魂 (Visual Portrait)**
  - **展示内容**：高质量角色立绘（Fullbody）。
  - **视觉效果**：带有呼吸灯效果的中心阴影，立绘略微超出容器边缘以增强立体感。

- **左上角 (Top-Left)：身份印记 (Identity Header)**
  - **对应详情页**：Basic Info Tab。
  - **展示内容**：角色名、种族、职业、性别、年龄标签。
  - **视觉风格**：精致的半透明卡片，带有边框发光。

- **右上角 (Top-Right)：能力矩阵 (Power Radar)**
  - **对应详情页**：Skills/Stats Tab (Radar Chart)。
  - **展示内容**：六维属性雷达图（STR, INT, AGI, VIT, DEX, LUK）。
  - **组件复用**：直接嵌入 `RadarChart` 组件，简化版交互。

- **左下角 (Bottom-Left)：叙事碎片 (Bio Fragment)**
  - **对应详情页**：Background/Story Tab。
  - **展示内容**：从 `brief_introduction` 中提取的一句感性引言。
  - **视觉风格**：优雅的斜体引用文字，配合手写体风格标题。

- **右下角 (Bottom-Right)：技能体系 (Skill Arsenal)**
  - **对应详情页**：Skills/Abilities Tab。
  - **展示内容**：最独特的 2-3 个技能图标（徽章）及名称。
  - **视觉风格**：横向排列的精美 Badge，带有动态悬停效果。

### 2. 交互与动效

- **动态切换**：支持左右箭头平滑切换不同角色，背景颜色随角色主题色（Theme Color）动态渐变。
- **详情下钻**：点击中央立绘或各个角落的“进入详情”图标，直接跳转至详情页对应的 Tab。
- **响应式适配 (Responsive)**：
  - **Desktop**：完整的四角布局，数据环绕立绘。
  - **Mobile**：立绘作为大背景，数据以纵向堆叠的半透明卡片组形式浮动在底部。

### 3. 数据驱动方案

- **数据获取**：复用 `findCharacterByUuid` 和 `parseCharacterModules` 逻辑。
- **配置式展示**：通过配置角色 UUID 列表，由服务端或 Client Hook 批量加载展示数据。
